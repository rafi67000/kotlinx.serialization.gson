# Kotlinx Serialization GSON Implementation

A simple implementation of a **GSON adapter** using **Kotlinx Serialization**, allowing you to serialize and deserialize Kotlin objects easily with GSON.

## Features

- Serialize Kotlin data classes to JSON using Kotlinx Serialization.
- Deserialize JSON into Kotlin objects seamlessly.
- Lightweight and simple to integrate into existing projects.
- Compatible with standard GSON usage patterns.

## Installation

Add the dependency to your `build.gradle.kts`:

```kotlin
repositories {
    maven("https://repo.rafi67000.xyz/releases/")
}

dependencies {
    implementation("xyz.rafi67000.kotlinx:kotlinx-serialization-gson:1.0.0")
}
```
